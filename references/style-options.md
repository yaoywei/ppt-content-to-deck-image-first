# Style Options - The 3 Starter Directions

When the user pastes a Chinese company brief and asks for a PPT, present these 3 styles as the default menu. Each fits a different audience. Always give the user a 1-line description + 1-line "best for" before they pick.

## A. 极简科技蓝 (Minimalist Tech-Blue)

**One-liner:** Deep navy gradient, gold accent line, single abstract drone icon, lots of negative space.

**Vibe:** Apple keynote meets McKinsey consulting deck. Restrained, professional, cool.

**Best for:**
- Government / 国企汇报
- State Grid / 南方电网 / 公安 system
- Formal investor pitch
- 任何"我需要看起来像央企/上市公司"的场合

**Risks:**
- 偏冷 - international / 东南亚客户可能觉得没温度
- 信息密度低 - 不适合要"塞 20 个数据点在一页"的场景

## B. 科技国潮金 (Tech Chinese-Gold)

**One-liner:** Deep midnight blue + metallic gold-foil Chinese calligraphy + silk-road / 1-belt-1-road visual motifs.

**Vibe:** Luxury Chinese tech brand. Silk Road meets SpaceX.

**Best for:**
- International gov-corp clients (一带一路叙事)
- 海外市场拓展 PPT
- 高端路演 / 政企合作
- "我想看起来像有中国底蕴 + 国际化"

**Risks:**
- 容易做俗 - 配色没控好就是乡镇企业家风
- 金色使用要克制 - 满屏金色 = 廉价

## C. 未来数据大屏 (Dark Command-Console)

**One-liner:** Near-black background, cyan / electric-blue holographic data streams, wireframe drone, Iron Man HUD feel.

**Vibe:** Iron Man HUD meets DJI Enterprise dashboard. High-tech, cinematic, investor-grade.

**Best for:**
- 投资人路演 (VC / PE)
- 互联网 / 科技公司客户
- "我的产品是技术 + 数据 + AI"
- 任何要"impressive demo" 的场合

**Risks:**
- 偏 cool - 对政府领导可能不够"正"
- 暗色底打印 / 投影可能看不清 (要确认会场是 LED 大屏还是投影)

## Hybrid Suggestions (when the user can't pick one)

| User wants | Suggested hybrid |
|---|---|
| 政企客户 + 投资人双线 | A as primary, C as "tech demonstration" insert slides |
| 国内 + 海外双线 | A for domestic, B for international |
| 科技公司 + 央企气质 | A as base, C for 1-2 "product demo" pages |
| 偏年轻 / 招聘场合 | A but with brighter accent color (teal / coral instead of gold) |

## What NOT to propose as a default style

- **孟菲斯 / Memphis** - too playful, doesn't fit corporate
- **Material Design 平面化** - dated since 2020
- **水墨 / 国画风** - looks great in still images, but text and data don't render
- **赛博朋克** - too edgy for any B2B context

The user can request these as a custom 4th option, but don't lead with them.
